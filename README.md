# 🚀 API de Predicción - "Niños Creativos"

Este proyecto implementa una **API (Application Programming Interface)** profesional para servir un modelo de Machine Learning entrenado. El objetivo es predecir una clasificación para un cliente potencial del servicio "Niños Creativos" basándose en su perfil demográfico y financiero.

> Este proyecto fue desarrollado siguiendo los estándares de **MLOps** para garantizar la reproducibilidad y el correcto manejo de dependencias entre entornos.

---

## 🛠️ Requisitos Previos

Debido a que el modelo fue entrenado con la versión moderna de **NumPy (2.0.2)** y **Scikit-learn (1.6.1)**, es **crítico** cumplir con lo siguiente:

* **Python:** Versión **3.10** o superior (Probado exitosamente en Python 3.12). Las versiones anteriores de Python (como 3.8 o 3.9) no permiten instalar las versiones requeridas de las librerías principales.
* **Git:** Instalado y configurado para control de versiones.
